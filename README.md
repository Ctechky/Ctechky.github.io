## Getting Started

1. **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/personal-blog.git
    ```

2. **Navigate to the project directory**:
    ```bash
    cd personal-blog
    ```

3. **Open the index.html file in your browser** to see the template in action.
